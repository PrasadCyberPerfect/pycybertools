# PyCyberTools — Frontend
Deploy this folder on Vercel.com (free)

## Before deploying
1. Deploy backend first on Render.com
2. Get your Render URL (e.g. https://pycybertools-api.onrender.com)
3. Open index.html and update line ~10:
   const API = "https://YOUR-RENDER-URL.onrender.com";

## Deploy on Vercel
1. Push this folder to GitHub
2. Go to vercel.com → Import repo → Deploy
3. Your site is live at: https://your-app.vercel.app
