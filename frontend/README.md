# 🛡️ PyCyberTools Frontend

A modern cybersecurity toolkit frontend built with HTML, CSS, and JavaScript.

The frontend provides a terminal-inspired user interface that connects to the PyCyberTools Backend API hosted on Render.

---

## 🚀 Live Architecture

Frontend (Vercel)
↓
Backend API (Render)
↓
Cybersecurity Tools Engine

---

## ✨ Features

- Modern Cybersecurity UI
- Terminal-style Interface
- Responsive Design
- Real-time API Integration
- Fast Deployment on Vercel
- Dark Hacker Theme
- Tool Result Viewer
- Error Handling
- Mobile Friendly
