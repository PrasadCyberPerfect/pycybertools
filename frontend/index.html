🚀 Deploy Backend on Render.com

The frontend runs on Vercel. The Python backend runs on Render.com (free). Follow these steps once:

1
Create GitHub repo for backend
Upload the backend/ folder to a new GitHub repo named pycybertools-api
2
Sign up at render.com
Go to render.com → Sign up free → New Web Service → Connect your GitHub repo
3
Configure Render settings
Render auto-detects everything. Just click Deploy. You'll get a URL like:
https://pycybertools-api.onrender.com
4
Update API URL in this file
Open frontend/index.html, find line ~10 and change:
const API = "https://pycybertools-api.onrender.com";
5
Deploy frontend to Vercel
Go to vercel.com → Import GitHub repo → Deploy the frontend folder. Done!
